## 7. Complete User Journey

1. **First-time User Journey**:
   - App Launch → Loading Screen → Login Screen → Register Screen
   - Complete Registration → Main Application
   - Request Microphone Permissions
   - Configure Emergency Keywords
   - Start Listening for Emergencies

2. **Returning User Journey**:
   - App Launch → Loading Screen → Login Screen
   - Sign In → Main Application
   - Start Listening for Emergencies

3. **Emergency Detection Journey**:
   - Listening → Emergency Keyword Detected
   - Emergency Alert Screen Displayed
   - Device Vibrates & Flashlight Activates
   - Emergency Logged to History
   - User Acknowledges Alert → Return to Listening

4. **Settings Configuration Journey**:
   - Navigate to Settings Tab
   - Adjust Alert Preferences (Flashlight, Vibration, Transcription)
   - Customize Emergency Keywords
   - Adjust Detection Sensitivity
   - View Emergency History
